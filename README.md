# Sales-and-Profit-Dashboard----PowerBI

# Description
Developed an interactive Sales Analysis Dashboard using advanced Microsoft Excel features to transform raw sales data into meaningful insights. Utilized Pivot Tables and complex formulas to accurately calculate key performance indicators such as Total Sales and Total Profit. Designed dynamic visualizations including bar, line, and donut charts that automatically update based on user selections. Implemented interactive slicers (Year, Region, Product Category) to enable real-time filtering and deeper data analysis, supporting better data-driven decision-making.
